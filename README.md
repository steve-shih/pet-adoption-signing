# 🐾 寵物領養簽署系統 (Professional Edition)

專業的寵物領養協議電子簽署系統，支持雙方簽名、多合約並行管理、自動快照紀錄、隱私保護與遠程分享。

## ✨ 核心特性

- ✅ **多合約並行管理** - 每個合約擁有獨立資料夾與 UUID，支持同時編輯多份合約。
- ✅ **雙方電子簽名** - 送養人和認養人各擁有獨立 Canvas 簽名區域。
- ✅ **分頁式操作介面** - 精簡的「建立新合約」、「合約列表」與「歷史存檔」分頁。
- ✅ **自動化存檔快照 (Snapshots)** - 每次存檔時自動生成完整合約截圖，記錄修改歷程。
- ✅ **分級密碼保護**：
    - **管理員權限**：執行刪除、修改隱私設定需經過管理員驗證。
    - **個別合約鎖定**：支持為特定合約設定專屬開啟密碼。
- ✅ **PNG 圖片導出** - 基於 html2canvas 的高品質合約導出。
- ✅ **數據管理** - 軟刪除機制，確保資料誤刪後仍可救回。

## 🛡️ 安全與隱私設計

### 管理員權限
系統設有管理員認證機制，執行敏感操作時會彈出驗證框：
- 啟用或更改合約的密碼鎖定。
- 從列表中刪除合約。
- 解鎖查看「歷史存檔」分頁。

### 合約專屬加密
針對需要保密的合約，管理員可以在「隱私權限管理」中為該合約設定**專屬開啟密碼**。設定後，任何人在列表點擊該合約時，都必須輸入正確的專屬密碼才能讀取內容。

## 🚀 快速開始

### 前置需求
- **Node.js** v16+
- **npm** v7+

### 安裝與啟動
1.  **安裝依賴**：
    ```bash
    npm install
    ```
2.  **本地啟動**：
    ```bash
    npm start
    ```
3.  **訪問應用**：
    打開瀏覽器訪問 `http://localhost:5000`

## 🔧 開發與部署建議

- **開發模式**：`npm run dev` (使用 nodemon 自動重載)。
- **遠程分享**：可搭配 `ngrok http 5000` 快速產生公網連結。
- **備份建議**：定期備份 `/data` 目錄以確保合約與簽名安全。
